== Description ==

WordPress changelog - logs any uploads, updates, installations/uninstallations, activations/deactivations of themes, plugins and WordPress core.
It could be helpful to easily find out the moment when something went wrong with your site and quickly fix the issue.

The plugin logs such actions:
* Theme installation
* Theme update
* Theme activation
* Theme deactivation
* Theme removal
* Plugin installation
* Plugin update
* Plugin activation
* Plugin deactivation
* Plugin removal
* Translation update
* WordPress update
* WordPress reinstall

== Installation ==

Installation

1. Upload the plugin files to the /wp-content/plugins/wp-changelog directory, or install the plugin through the WordPress plugins screen directly.
2. Go to Dashboard -> WordPress Changelog
